Caprice Tools for Sketchup

Collections of tools to make your Sketchup workflow easy

Caprice Tools offers a suite of powerful tools designed to make modeling in SketchUp easier and more efficient. Our current lineup includes the Angle Tool, Revolve Tool, Fast Loft Tool, and Shell Tool, among others. And this is just the beginning-more tools are coming soon.

Tools Introductions

Angle Tools

Select two connected edge, use the angle calculator to change the angle in degree between the selected edges.

Revolve Tool

A better alternative to SketchUpâ€™s built-in Follow Me tool, the Revolve Tool lets you easily revolve shapes around the X, Y, or Z axis. It even accounts for height variations, enabling the creation of more complex revolved forms.

Loft Tool

Quickly create a loft between two selected faces, with customizable control over the result using angle and normal strength parameters.

Shell Tool

Use Shell Tool to create models with consistent wall thickness. It allows you to turn any face of a solid geometry into a hollow shell with a specified thickness.
